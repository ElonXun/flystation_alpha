## flystation 
   该开源博客 基于react技术栈
   
   该项目由create-react-app脚手架创建
   
   线上地址: http://www.shangqingtian.cn:5000/#/

## 主要技术栈(前端)

   ```
     webpack3 
     react
     react-router 4.0
     redux
   redux-thunk
   Immutable
     antd
   ```
   
## 后端

    调[flystation_koa2](https://github.com/ElonXun/flystation_koa2)提供的接口
